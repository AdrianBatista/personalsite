How Background Removal Works

A practical overview of image segmentation and the workflow proposed by this tool.

GuideImage ProcessingWeb App

What is background removal?

Background removal separates the primary subject of an image from the surrounding pixels.

The task is more than a simple color replacement.

Typical processing stages

A removal service usually receives an image, identifies the foreground subject, and produces a mask.

The visual quality depends on the model and the input image.

Input

Validate the selected file.

Segmentation

Estimate the foreground pixels.

Mask

Create an alpha mask.

Output

Export a transparent image.

How to use this interface

The project currently provides the visual workflow only.

  1. Select an image.
  2. The file would be sent to the service.
  3. The result would be shown.
  4. Download the processed image.

The service can be connected later.
